Aqui vou falar sobre as tags HTML

<html> Define um inicio e um fim de um Documento HTML.

<head> Define informações sobre o documento, como título, meta tags e scripts.

<title> Define o título da página, que aparece na barra de título do navegador.

<body> Define o corpo do documento, onde todo o conteúdo é colocado.

<h1> Define os títulos das seções, do maior (h1) ao menor (h6).

<p> Define um paragráfo.

<a>Define um hyperlink para outra página ou recurso.

<img> Define uma imagem a ser exibida na págiina.

<ul> Define uma lista não ordenada.

<ol> Define uma lista ordenada.

<li> Define um item de lista.
